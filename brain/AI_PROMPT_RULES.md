# AI Prompt Rules

## Purpose
Capture project-specific rules for AI assistants.

## How To Use
- Follow these rules during planning, coding, and review.
- Update when the team changes collaboration expectations.

## Rules
- Keep the CLI macOS-friendly and shell-native.
- Do not add heavy dependencies without a clear reason.
- Preserve predictable command behavior over clever shortcuts.
- Document user-visible behavior in `README.md`.
- Record product or architecture decisions in `brain/decisions/`.

# Endpoints

## Purpose
Track public command entrypoints.

## How To Use
- Update when CLI commands change.

## Commands
- `termcode set <paths...>`
- `termcode ls`
- `termcode rename <current-name-or-path> <new-alias>`
- `termcode open <project>`
- `termcode . <project>`
- `termcode <project> <script> [args...]`
